JCLIF

Java Command Line Interface Framework

Introduction

JCLIF (Java Command Line Interface Framework) aims to provide a simple API which simplifies parsing of command line inputs passed on console based java applications. Framework also provides formatting of command line options usage patterns following Linux or Window specific help text formats.

Features

  • Simple API for specifying command line option and parameters programmatically or through annotations
  • Common interface for parsing windows or linux like command like inputs
  • Common interface for printing command line usage help text following windows or linux format
  • Provide parameter type validation for common input types (Strings, numbers, files)
  • API for customized parameter value parsing
